🔧 Quick Admin User Setup

⚠️ Important: The admin user doesn't exist yet

Follow these steps IN ORDER to create and login as admin.

Step 1: Register Admin User Through the App

  1. Open your app at: http://localhost:5175/#/register
  2. Fill in the registration form:
    • Email: admin@demo.com
    • Password: admin123
  3. Click Register
  4. You should see a success message

Step 2: Promote to Admin in Supabase

  1. Open Supabase Dashboard: Your Project
  2. Click SQL Editor in the left sidebar
  3. Click New query
  4. Copy and paste this SQL:
-- Promote admin@demo.com to admin role
UPDATE users 
SET role = 'admin', 
    status = 'active',
    updated_at = NOW()
WHERE email = 'admin@demo.com';

-- Add to user_roles table
INSERT INTO user_roles (user_id, role, assigned_at)
SELECT id, 'admin', NOW()
FROM users
WHERE email = 'admin@demo.com'
ON CONFLICT (user_id, role) 
DO UPDATE SET assigned_at = NOW();

-- Verify (should return 1 row with admin role)
SELECT 
    u.email, 
    u.role as users_role,
    ur.role as user_roles_role,
    u.status
FROM users u
LEFT JOIN user_roles ur ON ur.user_id = u.id
WHERE u.email = 'admin@demo.com';
  1. Click Run (or press F5)
  2. You should see a result showing the admin user with role='admin'

Step 3: Login as Admin

  1. Go back to your app: http://localhost:5175/#/login
  2. Login with:
    • Email: admin@demo.com
    • Password: admin123
  3. You should now see:
    • ✅ "Admin" badge in the navbar
    • ✅ "Admin Panel" link in the navigation
  4. Click "Admin Panel" to access the admin dashboard!

✅ Success Checklist

  • ☐ Registered admin@demo.com through the app
  • ☐ Ran the SQL script in Supabase
  • ☐ SQL returned 1 row with admin role
  • ☐ Logged in successfully
  • ☐ Can see "Admin Panel" link
  • ☐ Can access /admin page

🐛 Troubleshooting

"Invalid login credentials" - Make sure you registered first through the app

"No Admin Panel link" - Run the SQL script again, then logout and login

"Access denied to /admin" - Check that the SQL script ran successfully
